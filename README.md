# Anshell

Anshell is a personal web application designed to organize and browse collections of anime, manga, TV series, books, and video games.

The project is built as a static website using HTML, CSS, and vanilla JavaScript, with all media information stored in JSON files. It focuses on providing a simple, fast, and responsive browsing experience.

This repository is used exclusively to host the website through GitHub Pages.

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* JSON
